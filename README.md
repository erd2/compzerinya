# Telegram-бот мониторинга фриланс-бирж

Асинхронный Telegram-бот на Python для мониторинга заказов на **Kwork**, оценки релевантности через OpenRouter AI, генерации откликов и отправки уведомлений.

## Как это работает

1. **Парсинг** — Selenium загружает Kwork, открывает каждую карточку заказа и забирает полное описание
2. **Оценка** — OpenRouter AI (Gemini) оценивает релевантность заказа вашим навыкам (0-100%)
3. **Отклик** — AI генерирует персональный текст отклика для каждого релевантного заказа
4. **Уведомление** — заказ + описание + готовый отклик отправляются в Telegram
5. **Хранение** — все заказы сохраняются в SQLite, дубликаты пропускаются

## Структура

```
├── main.py                 # Точка входа, бот + планировщик + /cover
├── config.py               # Загрузка настроек из .env
├── db.py                   # SQLite CRUD
├── scheduler.py            # Цикл опроса бирж
├── notifier.py             # Отправка Telegram-уведомлений
├── llm.py                  # Оценка релевантности + генерация отклика
├── exchanges/
│   ├── kwork.py            # Парсер Kwork (Selenium, переход на страницу заказа)
│   └── yandex_uslugi.py    # Парсер Яндекс.Услуг (Selenium)
├── models/
│   └── order.py            # Модель заказа
├── utils/
│   ├── helpers.py          # Очистка текста
│   ├── cache.py            # TTL-кэш для LLM
│   ├── validators.py       # Валидация цены и URL
│   └── selenium_login.py   # Авторизация Kwork через Selenium
└── middleware/
    └── rate_limiter.py     # Rate limiting для запросов
```

## Быстрый старт

1. Скопируйте `.env.example` в `.env`
2. Заполните токены и настройки
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустите бота:
   ```bash
   python main.py
   ```

## Настройки (.env)

| Переменная | Описание |
|---|---|
| `TELEGRAM_BOT_TOKEN` | Токен Telegram-бота от @BotFather |
| `TELEGRAM_CHAT_ID` | Ваш ID для уведомлений (узнать через @userinfobot) |
| `OPENROUTER_API_KEY` | Ключ OpenRouter для оценки заказов |
| `MY_SKILLS` | Ваши навыки через запятую |
| `RELEVANCE_THRESHOLD` | Минимальная релевантность для уведомления (50-100) |
| `POLL_INTERVAL_SECONDS` | Интервал опроса бирж в секундах |
| `KWORK_COOKIES` | Cookies Kwork для авторизации |
| `KWORK_USE_SELENIUM` | Использовать Selenium для Kwork (true/false) |
| `KWORK_LOGIN` / `KWORK_PASSWORD` | Логин/пароль Kwork (если нет cookies) |
| `SELENIUM_HEADLESS` | Запускать браузер в фоне (true/false) |

## Команды бота

| Команда | Описание |
|---|---|
| `/start` | Информация о боте и список команд |
| `/stats` | Последние 5 заказов из БД с релевантностью |
| `/cover <ссылка>` | Сгенерировать отклик для любого заказа по ссылке |
| `/stop` | Остановка бота |

### Пример /cover

```
/cover https://kwork.ru/projects/3148645
```

Бот откроет страницу заказа, извлечёт описание и сгенерирует готовый отклик под ваши навыки.

## Уведомление в Telegram

Каждое уведомление содержит:
- 📌 Тему заказа
- 📊 Процент релевантности
- 🔗 Ссылку на заказ
- 📋 Полное описание
- 📝 Готовый текст отклика (скопируйте и отправьте заказчику)

## Известные проблемы

- **TelegramConflictError** — не запускайте несколько копий бота. Проверьте: `tasklist | findstr python`
- **Kwork cookies** — периодически протухают, обновите из браузера
